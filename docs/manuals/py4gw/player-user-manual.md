# Player User Manual

`GW::player` manages selected player-facing actions and lookups.

Main capabilities:

- active title control
- player-name lookup and replacement
- player-agent lookup
- profession/faction actions

Representative API:

- `SetActiveTitle(...)`
- `RemoveActiveTitle()`
- `GetPlayerAgentId(...)`
- `GetPlayerName(...)`
- `SetPlayerName(...)`
- `ChangeSecondProfession(...)`
- `DepositFaction(...)`

Use it for player identity and title/faction actions.
