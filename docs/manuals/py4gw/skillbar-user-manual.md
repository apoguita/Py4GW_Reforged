# Skillbar User Manual

`GW::skillbar` manages skill usage, skill templates, and attribute loading.

Main capabilities:

- use skills
- load skillbars and templates
- set attributes
- change secondary profession
- query unlocked/learnt skills
- register use-skill callbacks

Representative API:

- `UseSkill(...)`
- `UseSkillByID(...)`
- `DecodeSkillTemplate(...)`
- `EncodeSkillTemplate(...)`
- `LoadSkillTemplate(...)`
- `SetAttributes(...)`

Use it for skillbar automation and skill template workflows.
