# Guild User Manual

`GW::guild` provides guild-hall and player-guild convenience methods.

Main API:

- `GetPlayerGuildIndex()`
- `GetPlayerGuildAnnouncement()`
- `GetPlayerGuildAnnouncer()`
- `TravelGH()`
- `TravelGH(Context::GHKey key)`
- `LeaveGH()`

Use it for guild-hall travel and simple player-guild data lookups.
