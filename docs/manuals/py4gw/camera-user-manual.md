# Camera User Manual

`GW::camera` controls runtime camera movement and selected camera patches.

Main API:

- movement helpers such as `ForwardMovement`, `SideMovement`, `VerticalMovement`, `RotateMovement`
- settings helpers such as `SetMaxDist`, `SetFieldOfView`
- patch/state helpers such as `UnlockCam`, `GetCameraUnlock`, `SetFog`

Use it for camera manipulation and camera-related quality-of-life toggles.
