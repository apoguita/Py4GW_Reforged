# Party User Manual

`GW::party` manages party composition, hard mode, hero control, pet behavior, and party search.

Main capability groups:

- party state queries
- leader and load-state helpers
- hero and henchman management
- player invitation and removal
- hero/pet flagging and behavior
- party search and replies

Representative API:

- `get_party_size()`
- `set_hard_mode(...)`
- `add_hero(...)`
- `kick_hero(...)`
- `invite_player(...)`
- `flag_hero(...)`
- `flag_all(...)`
- `set_hero_behavior(...)`
- `search_party(...)`

Use it for party operations and hero control.
