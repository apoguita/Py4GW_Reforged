# Trade User Manual

`GW::trade` handles player-to-player trade actions.

Main API:

- `OpenTradeWindow(...)`
- `AcceptTrade()`
- `CancelTrade()`
- `ChangeOffer()`
- `SubmitOffer(...)`
- `RemoveItem(...)`
- `OfferItem(...)`

Use it for normal trade flows rather than building trade packets manually.
